# SkillSwap - Peer Learning & Mentorship Platform

A full-stack web application built with the PERN stack (PostgreSQL, Express, React, Node.js) that enables users to exchange knowledge and skills through peer-to-peer learning.

## Features

- User authentication and authorization
- Profile management
- Skills marketplace (offer and request skills)
- Mentorship request system
- Personalized dashboard

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Authentication:** JWT with bcrypt
